# React-work-test

## Overview

React + TypeScript を用いたフロントエンドのワークテスト・課題演習用リポジトリ。

## Purpose

- React Hooks / 状態管理の理解深化
- 実務を想定したコンポーネント設計
- ワークテスト・アウトプット整理

## Learning Method

- 課題は実務を想定したテーマで出題
- 自身で設計・実装を行った上で、生成AIとの壁打ちによるレビュー・評価を実施
- 指摘内容を元に改善・リファクタリングを行い、理解を深める学習サイクルを採用

## Tech Stack

- React
- TypeScript
- Vite
- ESLint / Prettier

## Project Structure

- `src/work-sample/`
  - 日付・課題ID単位での課題管理

## Development

```bash
npm install
npm run dev
```
