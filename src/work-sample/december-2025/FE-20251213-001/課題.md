# useEffect 初級：疑似 API フェッチでローディング表示

- 課題 ID：20251213-001
- 難易度：初級（肩慣らし）

## 課題内容

以下の条件を満たす 小さな React コンポーネント を作ってください。

### 要件

1. コンポーネント名：FakeApiLoader
2. マウント時に useEffect を使って 2 秒後にデータが取得できたように見せる
   - setTimeout を使う
   - データは { message: "Hello, React!" } を返した体でOK
3. データ取得中は「Loading...」 と表示する
4. データ取得後は取得したメッセージを表示する
   - `Data: Hello, React!`
5. TypeScript で型定義をつけること（初級なので簡単でOK）

### 提出物

- コンポーネントのコード（React + TypeScript）
- 動作の簡単な説明（1〜2文でOK）

---

> 動作の簡単な説明（1〜2文でOK）

- propsは初期値としてのみ使用
- stateはコンポーネント内部で管理

`const [data, setData] = useState<string>(initialData)`

- setTimeout()の戻り値（タイマーID）をtimer変数に代入。これは後でタイマーを止めるために使う。

```jsx
const timer = setTimeout(() => {
  setData('Hello, React!')
}, 2000)
```

- クリーンアップ関数（コンポーネント型が画面から消えるときや、useEffectが再実行される前に呼ばれる）
- タイマーを止めて、余計な処理が走らないようにする

`return () => clearTimeout(timer) `
