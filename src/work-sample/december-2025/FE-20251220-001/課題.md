# useState 超初級：数値を表示するだけ

- 課題 ID：FE-20251220-001
- 難易度：超初級（完全に肩の力を抜く）

## 課題内容

数値の state を作って、そのまま表示するだけのコンポーネントを作ってください。
操作・イベントは一切不要。

### 要件

1. コンポーネント名：NumberStateView
2. `useState<number>` を使う
3. 初期値は任意の数値（例：0 や 100）
4. state の値を JSX で表示する `value: 100`
5. setStateは使わない

### 提出物

- コンポーネントのコード
- 説明不要

### この課題の狙い（軽く）

- useState は「変える予定がなくても、render に乗せたい値」を持てる
