# seState 超初級：初期値を表示するだけ

- 課題 ID：FE-20251217-005
- 難易度：超初級（理解を固定する用）

## 課題内容

何もしなくても表示される state を作ってください。
クリックも入力も不要。

### 要件

1. コンポーネント名：InitialStateView
2. `useState<string>` を使う
3. 初期値は好きな文字列でOK（例："Hello useState"）
4. state の値をそのまま画面に表示する
5. setState は 一切使わない

### 提出物

- コンポーネントのコード
- 説明不要

### この課題の“隠れテーマ”

- useState は「変更するため」だけのものじゃない
- 初期値 = そのコンポーネントのデフォルト状態
- 実務だと以下に直結する
  - API 初期値
  - フォーム初期表示
  - フラグの初期状態
