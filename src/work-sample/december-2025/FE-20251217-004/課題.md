# JSX 初級：条件付きレンダリングを書くだけ

- 課題 ID：FE-20251217-004
- 難易度：超初級（脳を使わない）

## 課題内容

true / false を直接 JSX に書いて
表示・非表示を切り替えるだけのコンポーネントを作ってください。
（useState は使わない）

### 要件

1. コンポーネント名：SimpleCondition
2. コンポーネント内に以下の定数を置く
   - `const isVisible = true`
3. isVisible が true のときだけ「表示されています」を表示する
4. false にしたら表示されないこと

### 提出物

- コンポーネントのコードだけでOK
- 説明不要

### この課題の狙い（軽く）

- uJSX の中で 条件を書く感覚 を維持する
- state に頼らない表示制御を思い出す
