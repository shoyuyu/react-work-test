# useEffect 初級：マウント時にログを出す

- 課題 ID：FE-20251217-003
- 難易度：初級（副作用の入口）

## 課題内容

コンポーネントが表示された瞬間（初回マウント時）にだけ
console.log を出すコンポーネントを作ってください。

### 要件

1. コンポーネント名：MountLogger
2. useEffect を使用する
3. コンポーネントの初回表示時にだけ、以下をログ出力する
   - `mounted`
4. 再レンダーでは ログが増えない こと
5. TypeScript で書く（型はシンプルでOK）

### 提出物

- コンポーネントのコード
- 「なぜ1回しか呼ばれないか」を1文で説明

### この課題で見ているポイント

- useEffect と依存配列の理解
- 「レンダー」と「副作用」を分けて考えられているか
- 無意味な state を作っていないか

### この課題の狙い（重要）

- useEffect は イベントでも state でもない ことを体感する
- 「いつ実行されるか」を言語化できるようになる

---

## 提出物の問いへの解答

> 「なぜ1回しか呼ばれないか」を1文で説明

useEffectの依存配列を空配列にしているため。
