# props 初級：親から受け取った文字列を表示する

- 課題 ID：FE-20251217-001
- 難易度：初級（useState 以前の基礎固め）

## 課題内容

親コンポーネント → 子コンポーネント に
props で文字列を渡して表示するだけの構成を作ってください。

### 要件

1. 子コンポーネント
   - コンポーネント名：Message
   - props で text を受け取る
   - 受け取った text をそのまま画面に表示する
   - TypeScript で props の型を定義する
   - `<Message text="Hello Props!" />`
2. 親コンポーネント
   - Message コンポーネントのコード
   - 親コンポーネント側の呼び出しコード
   - props の型をどう定義したか（1文でOK）

### この課題で見ているポイント

- props と state を混同していないか
- 型定義がシンプルで過不足ないか
- コンポーネントの責務が分かれているか

### 提出物

- Message コンポーネントのコード
- 親コンポーネント側の呼び出しコード
- props の型をどう定義したか（1文でOK）

---

## 提出物の問いへの解答
