# useState 初級：表示／非表示を切り替える

- 課題 ID：FE-20251214-001
- 難易度：初級（肩慣らし）

## 課題内容

### 要件

1. コンポーネント名：`ToggleMessage`
2. ボタンを1つ表示する
   - 初期表示テキスト：`Show Message`
3. ボタンをクリックすると
   - メッセージ表示：`Hello, React + TypeScript!`
   - ボタン文言：`Hide Message`
4. もう一度クリックすると
   - メッセージは非表示
   - ボタン文言は `Show Message` に戻る
5. 状態管理は `useState<boolean>` を使う
6. TypeScript の型は明示する（簡単でOK）

### 提出物

- コンポーネントのコード（React + TypeScript）
- どの state をどう使っているかの簡単な説明（1文でOK）

---

## 提出物の問いへの解答

> どの state をどう使っているかの簡単な説明

stateはboolean型で「表示/非表示」を管理

`const [isVisible, setIsVisible] = useState<boolean>(false)`
