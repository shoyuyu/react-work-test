# useState 初級：入力した文字をリアルタイム表示する

- 課題 ID：FE-20251217-002
- 難易度：初級（state の役割を身体に入れる）

## 課題内容

input に入力した文字列を、同じ画面内にリアルタイムで表示する
シンプルなコンポーネントを作ってください。

### 要件

1. コンポーネント名：TextPreview
2. `<input type="text" />` を1つ表示する
3. input に入力した文字を `useState<string>` で管理する
4. 入力中の文字列を下にそのまま表示する
5. TypeScript の型は明示する（初級なのでシンプルでOK）

### 提出物

- コンポーネントのコード
- state の型と役割を1文で説明

### この課題で見ているポイント

- onChange と event.target.value の扱い
- state を 表示のためだけに使っている
- 不要な処理を書いていないか

### この課題の狙い（重要）

- state = 画面を変えるための唯一の情報源 だと体感する
- フォーム系実装の最小単位を押さえる
